### App1: min-Todo
A lightweight, server-side task manager built with Flask and vanilla JavaScript.

**Technology Stack:**
- Python Flask (Backend)
- HTML/CSS/JavaScript (Frontend)
- Server-side rendering

**Features:**
- Basic task creation and deletion
- Simple task list management
- Minimal UI with essential functionality
- Lightweight and easy to deploy

**Getting Started:**
```bash
cd app1
pip install -r requirements.txt
python app.py
```
Visit `http://localhost:5000`
